# jianshu

简书有一个漏洞，可以无限刷阅读量。这个漏洞很有意思，非登录状态下，打开一次或刷新一次文章阅读量就加一次。

只需替换代码中的Post_link为你的文章地址即可

代码默认刷100次即退出
